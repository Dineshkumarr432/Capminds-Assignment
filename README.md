# Capminds-Assignment
Responsive Appointment Scheduling application

🗓️ Appointment Scheduling Application
📌 Project Overview

This is a responsive Appointment Scheduling web application built using HTML, CSS, and JavaScript (Vanilla JS).

The application allows users to:

Book appointments
View them in a calendar
Edit/Delete appointments
Search and filter appointments

⚠️ Note: No frameworks or external libraries were used (pure frontend implementation).

🚀 Features
✅ Appointment Management
Book new appointments using a popup form
Edit existing appointments
Delete appointments
Instant UI updates without page refresh
📅 Calendar View
Displays appointments based on selected date
Shows:
Patient Name
Doctor Name
Time
Updates dynamically when appointments are added/edited/deleted
🔍 Search & Filter
Search by:
Patient Name
Doctor Name
Filter by:
Date
Date Range
Real-time filtering
📱 Fully Responsive Design

Works across:

Mobile 📱
Tablet 📲
Desktop 💻
🛠️ Tech Stack
HTML5 – Structure
CSS3 – Styling & Responsive Design
JavaScript (ES6) – Logic & Functionality
LocalStorage – Data persistence
